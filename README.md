## Description
A MongoDB-based restaurant ordering system built using:
- Node.js
- Express
- MongoDB
- HTML, CSS, JavaScript

## Features
- View restaurant menu 
- Place food orders
- Live order total calculation
- Order status tracking
- Admin order management
- Receipt generation

## How to Run
1. Start MongoDB locally
2. Run `node server.js`
3. Open `frontend/index.html` in browser

## Database
MongoDB (Local)
Database Name: restaurantDB
